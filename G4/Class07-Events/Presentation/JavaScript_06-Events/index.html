Event listeners

  • Allow one event to trigger multiple functions
  • Just a function that sits there and waits for an event to happen

Event listeners


const usernameInputEl = document.getElementById("username");
usernameInputEl.addEventListener("blur", function(){
    const greeting = `Hello ${usernameInputEl.value}`;
    console.log(greeting);
});

function greetingFunc(){
    const greeting = `Hello ${usernameInputEl.value}`;
    console.log(greeting);
}
usernameInputEl.addEventListener("blur", greetingFunc);

USING PARAMETERS WITH EVENT HANDLERS & LISTENERS

  • If you need to pass arguments to a function that is called by an event handler or listener, you wrap the function call in an anonymous function


//Get username input
const usernameInputEl = document.getElementByid('username');
// Get feedback element 
const feedbackEl = document.getElementByid('feedback');

function validateUsernameLength(minlength) {// Declare function 
	//If username too short 
	if(usernameInputEl.value.length < minlength){
		// Set the error message
		feedbackEl.textContent = `Username must be ${minlength} characters or more`;
	}else{ // Otherwise
		// Clear msg
		feedbackEl.textContent = "";  
	}
}
//We add an event listener when input loses focus that validates it using anonymous function that calls the validator inside
usernameInputEl.addEventListener('blur',function(){
// Pass arguments here 
	validateUsernameLength(5); 
}, false);

  • Finaly print the string in a new p element on the screen

Student Exercise - #3 Solution




	
	
	
	





						


const inputs = document.querySelectorAll("input")
const resultEl = document.getElementById("result");
const submitBtnEl = document.getElementById("submit");

function getUserInfo(inputElements){
	let inputValues = ""
    for (let input of inputElements) {
        inputValues += input.value + ", ";
    }
    const result = `User: ${inputValues} is registered!`
    return result;
}
submitBtn.addEventListener("click", function(){
    result.innerText = getUserInfo(inputs);
})
