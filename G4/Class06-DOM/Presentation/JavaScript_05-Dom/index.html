Introduction (1)

  • DOM: Document Object Model
  • Structured representation of an HTML document
  • Specifies how browsers should create a model of an HTML page
  • JavaScript can interact with it in order to access and update the contents of a web page while it is in the browser window by using the document object

Introduction (2)

  • The model is stored in the browsers memory
  • It consists of four main types of nodes
  • Each node is an object with methods and properties
  • Scripts access and update this DOM tree (not the source HTML file)
  • Any changes made to the DOM tree are reflected in the browser

How does it work?

  1. Decide on a webpage to write
  2. Write down the HTML
  3. Write some JavaScript
  4. Open it in a browser
  5. Browser parses the code and creates the DOM
  6. Manipulate the DOM with JavaScript when needed
  7. Finally learn how to merge javascript with html

Where does the DOM exist

  • IT EXISTS IN THE BROWSER
  • Changes on the DOM are only reflected in the browser

THE DOCUMENT NODE

  • At the top of the tree a document node is added
  • It represents the entire page
  • It is the starting point for all visits to the DOM tree

ELEMENT NODES

  • HTML elements describe the structure of an HTML page
  • To access the DOM tree, you start by looking for elements. Once you find the element you want, then you can access its text and attribute nodes if you want to.
  • Relationships between the document and all of the element nodes are described using the same terms as a family tree: parents, children, siblings, ancestors, and descendants. (Every node is a descendant of the document node.)

ATTRIBUTE NODES

  • The opening tags of HTML elements can carry attributes and these are represented by attribute nodes in the DOM tree
  • Attribute nodes are not children of the element thar carries them, they are part of that element
  • Once you access an element, there are specific JavaScript methods and properties to read or change that element's attributes.

TEXT NODES

  • Once you have accessed an element node, you can then reach the text within that element which is stored in its own text node
  • Text nodes cannot have children
  • Once you access an element, there are specific JavaScript methods and properties to read or change that element's text.

Working with DOM

How to get an element node?

  • Accessing and updating the DOM tree involves two steps:
    1. Locate the node that represents the element you want to work with
    2. Use its text content, child elements, and attributes

SELECTORS

  • Locating and selecting the node that you need is done with selectors
  • Selectors are methods on the document node that can find some element in the document
  • There are different types of selectors

Selecting by Id

Selector that selects items by the id attribute

Element can have only one unique id in the whole document

getElementById always returns a single element


// Selecting an element by Id and adding it to a variable for later use
let mainHeadingEl = document.getElementById("mainTitle");
console.log(mainHeadingEl); // Will give us the whole element node
console.log(mainHeadingEl.text); // Will give us the text inside the element

Selecting by Class

Selector that selects items by the class attribute

Element can have multiple classes and they don't have to be unique

getElementsByClassName always returns a collection of elements


// Selecting element by class and adding them to a variable for later use
let paragraphs = document.getElementsByClassName("paragraph"); 
let secondParagraph = document.getElementsByClassName("second"); 
console.log(paragraphs); // Will give us an array with 2 paragraphs
console.log(paragraphs[0]); // Will give us the first paragraph
console.log(secondParagraph); // Will give us an array with 1 paragraph
console.log(secondParagraph[0]); // Will give us the only paragraph

Selecting by Tag Name

Selector that selects items by the Tag Name

getElementsByTagName always returns a collection of elements


// Selecting elements by tag name and adding them to a variable for later use
let paragraphs = document.getElementsByTagName("p"); 
let texts = document.getElementsByTagName("text"); 
console.log(paragraphs); // Will give us an array with 2 paragraphs
console.log(paragraphs[0]); // Will give us the first paragraph
console.log(texts); // Will give us an array with 1 text
console.log(texts[0]); // Will give us the only text

Query Selectors

Selectors that select any element by CSS selector

querySelector always returns the first element it finds

querySelectorAll always returns a collection of elements


// Selecting elements by tag name and adding them to a variable for later use
let paragraphs = document.querySelectorAll("p"); 
let paragraphEl = document.querySelector(".paragraph");
let mainHeadingEl = document.querySelector("#mainTitle");
console.log(paragraphs); // Will give us an array with 2 paragraphs
console.log(paragraphEl); // Will give us the first paragraph it finds
console.log(mainHeadingEl); // Will give us the first header it finds

Traversing through the DOM tree

  • Selected elements give us the power to select other elements near them
  • We can select all the siblings, all child elements or get the parent element

Finding sibling elements

  • Sibling elements are all elements that have the same parent
  • nextElementSibling property selects the next sibling
  • previousElementSibling property selects the previous sibling

    • 
// Getting all elements with class myParagraph in an array and selecting the first element
let paragraph = document.getElementsByClassName("myParagraph")[0];
let sibling = paragraph.nextElementSibling; // We add the next sibling element here
console.log(paragraph); // The first paragraph
console.log(sibling); // The second paragraph
console.log(sibling.previousElementSibling); // Will log the first paragraph again

Finding the parent element

  • Every element has a parent element that we can find
  • parentElement property gives us the parent of the current element

    • 
// Getting all elements with class myParagraph in an array and selecting the first element
let paragraph = document.getElementsByClassName("myParagraph")[0];
let parentDiv = paragraph.parentElement; // We take the parent of the paragraph
console.log(paragraph); // The first paragraph
console.log(parentDiv); // The parent div with the class .container

Finding child elements

  • Child elements are all elements that are nested inside another element
  • We can select all children, first or last child of an element

    • 
// Getting all elements with class myParagraph in an array and selecting the first element
let mainDivEl = document.getElementById("main");
let mainDivChildren = mainDivEl.children; // All children of mainDivEl
let mainDivFirstChild = mainDivEl.firstElementChild; // Get first child of mainDivEl
let mainDivLastChild = mainDivEl.lastElementChild; // Get last child of mainDivEl
console.log(mainDivChildren); // Array of all children of mainDivEl
console.log(mainDivFirstChild); // The first child element of mainDivEl
console.log(mainDivLastChild); // The last child element of mainDivEl

Our DOM and relations

Changing content of an element

  • We can also change the content of an element
  • We can use different tools for different content

Getting text

  • We can get or change a text of an element
  • innerText gets or changes only the text without white spaces
  • textContent gets or changes only the text with white spaces
  • If used on parent it will get all text of all child nodes as well

    • 
let mainDivEl = document.getElementById("main");
let mainDivTextContent = mainDivEl.textContent; // Get content with white spaces
let mainDivInnerText = mainDivEl.innerText; // Get content without white spaces
console.log(mainDivTextContent);
console.log(mainDivInnerText);

Changing text in an element

  • Changing an element text is done by editing it as a string
  • We can treat the text property as a string, concatanate it and remove it

    • 
let mainHeadingEl = document.getElementById("mainTitle");
console.log(mainHeadingEl.innerText); // Checking what the text is
mainHeadingEl.innerText = ""; // Changing the text to nothing
console.log(mainHeadingEl.innerText);
mainHeadingEl.innerText = "New Page Title";
console.log(mainHeadingEl.innerText);
mainHeadingEl.innerText += "Newest Page Title";
console.log(mainHeadingEl.innerText);

Changing and adding code
